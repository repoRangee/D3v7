# Essex PBI Base

A collection of libraries to assist in the creation of PowerBI custom visuals.

# Building
* Install [Yarn](https://yarnpkg.com)
* Run `yarn` to install the dependencies
* Run `yarn test` to symlink, build, lint, and test all of the projects within the packages folder

# Projects

* [Visual Utils (visual-utils)](packages/visual-utils) - A collection of utilities the development of powerbi custom visuals.
* [Visual Settings (visual-settings)](packages/visual-settings) - A library to parse and enumerate powerbi visual settings.
* [Visual Testing Tools (visual-testing-tools)](packages/visual-testing-tools) - A collection of testing tools for unit testing powerbi custom visuals.
* [Visual Styling (visual-styling)](packages/visual-styling) - A collection of styling elements for producing a consistent powerbi visual experience.
